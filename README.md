# Personal Online CV Website

## Overview

This project is a personal online CV website developed using HTML5, CSS3 and Bootstrap. The website serves as an interactive curriculum vitae, presenting my education, work experience, technical skills and software development projects in a clean, responsive and professional format.

The website has been designed to showcase both my professional background as an educator and my transition into software engineering through the HyperionDev Software Engineering Bootcamp.

## Features

* Responsive webpage built with Bootstrap 5
* Navigation bar with links to each section of the page
* Professional header containing contact information
* Personal profile and career summary
* Skills and competencies section
* Education history
* Work experience
* Leadership and achievements
* Featured software development projects
* Responsive card-based layout
* Custom CSS styling

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
